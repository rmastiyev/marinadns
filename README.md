# MarinaDNS

**DNS & network toolkit for sysadmins, DevOps, and infosec professionals.**

🌐 **[marinadns.io](https://marinadns.io)**

## Tools (40+)

- DNS lookup, propagation checker, WHOIS/RDAP
- RBL/DNSBL blacklist check
- TLS/SSL grader, HTTP headers, DMARC/SPF/DKIM
- IP geolocation, ASN lookup, traceroute, ping
- Generators, converters, and more

## Stack

Nginx · PHP 8.3 · Unbound · SQLite · Cloudflare

## Languages

Available in English, Russian (RU), and Turkish (TR).
